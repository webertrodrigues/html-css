Principais formatações

Negrito

Nessa frase temos um termo em destaque usando o <strong>.


Itálico

Nesta frase, temos um termo em itálico usando a tag <em>.


Texto marcado (marcador de texto)

Também podemos criar textos marcado usando a tag <mark>.


Texto grande e pequeno

Estamos criando uma texto grande e uma texto pequeno.

A tag <big> é uma tag obsoleta (não é recomendada), porém a tag <small> ainda pode ser utilizada, pois ainda tem sentido (HTML), enquanto a <big> só tem forma (CSS).


Texto deletados

Podemos marca um texto com excluido para indica que ele deve ser lido, porem não considerado.


Texto inserido / sublinhado

Podemos marca um texto como inserido utilizando a tag <ins>.


Texto sobreescritos

Para coisas como X20 - 3, utilizamos a tag <sup> para que o 20 fique como espoente


Tetxo subescritos

Para coisas como H2O, usamos a tag <sub> para para elevar o 2 sobre o H
